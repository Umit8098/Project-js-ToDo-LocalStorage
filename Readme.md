# 📝 ToDo App

Kullanıcılar görev ekleyebilir, tamamlandı olarak işaretleyebilir ve silebilir.

## ✨ Özellikler
- ✅ Yeni görev ekleme  
- ✅ Görevleri tamamlandı olarak işaretleme  
- ✅ Görevleri silme
